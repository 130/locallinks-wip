whitelist
address handled schemes
* file://

It mainly allows opening file:// links on pages loaded by http(s):// scheme.

Whitelist interface

Hotkeys:

  • Enter/Insert - add rule
  • Enter (in rule editing) - save rule
  • Esc (in rule editing) - cancel rule changes

Left click on rule - edit rule.

Alt + Left click on rule - delete rule.

Holding left mouse button on rule and moving mouse around - reorder rules.
